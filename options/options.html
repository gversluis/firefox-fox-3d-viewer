Permissions
To detect requests to 3D files and to catch blobs (in memory files) this extension needs access to all pages.

You can disable this if you only want to upload local files.
If you disable this and want to view extensions which require a stub URL, then do not forget to grant access to the stub URL at the bottom of this page.
Display 3D files with the selected extensions
The selected file types will be displayed using three.js loaders.
Stub page
Extensions are limited by a content security policy.
This means extensions can not load on the fly generated scripts which some loaders use.
Another website is loaded as workaround. The extension is inserted so it loads as a regular page.
Use stub for 3D files with the selected extensions
These Three.js loaders use generated scripts.
Stub URL
The stub URL needs to be an existing URL which will be loaded before showing the 3D file. It won't affect browsing to the page itself.
URL https://www.example.org/ should be private since it is not supposed to do any tracking.
